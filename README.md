### This is a free Dukaan dashboard ui kit built with:

-   React
-   Tailwind CSS
-   Recharts
-   Framer Motion

It is completely responsive and can be used for any type of web application.

### Run this app locally

```shell
npm install
```

```shell
npm run start
```
